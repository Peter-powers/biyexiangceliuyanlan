虽然阿里的新一代语音合成技术kan-tts，让合成定制成本降低10倍以上。目前已经向b端客户开放，且后期个人用户也可只需10分钟手机录音，就可以定制ai录音。但是目前也没有看到它开始进行开源的行为。所以未来，科大讯飞所提供的语音合成api所被挑战的难度还是很大的。

至于像图像识别还是会被face++所替代。

可由于微软收费更加便宜且支持免费，虽然前期效果不如人意，但一旦数据量大了，我们就可以使用自己的数据，利用微软的算法来进行语音合成。
